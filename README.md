# Data Insert Tools

## Overview

The Data Insert Tools App is a GUI application written in wails (Go framework) that allows users to insert data into database table.

> Note: This app only test the availability of postgresql.

## Features:

- Flexible Configuration: Easy connect to several database (postgresql, mysql, sqlite, mongodb)

## Prerequisites

Need to install [wails](https://wails.io/).

## Installation

1. Clone the Repository:

```bash
git clone https://github.com/tckeong/db-insert-tools.git
cd data-insert-tools
```

2. Run the projects

```bash
wails dev
```
